# 🎲 Estatística de Dados RPG

Aplicação web simples para simular rolagens de dados de RPG e visualizar a probabilidade de sucesso com base em diferentes configurações.

## Sobre o projeto

Este app foi criado para testar rolagens de dados de forma prática e visual, permitindo simular várias tentativas e observar a taxa de acerto, distribuição dos resultados e comparação entre sucessos e falhas.

A proposta é oferecer uma ferramenta fácil de usar para jogadores, mestres e criadores de sistemas de RPG que queiram analisar mecânicas de forma mais objetiva.

## Funcionalidades

- Simulação de rolagens com múltiplos dados
- Escolha da quantidade de lados do dado
- Definição da dificuldade alvo
- Aplicação de bônus
- Modos de rolagem:
  - **soma**: soma todos os dados
  - **melhor**: usa o maior valor rolado
  - **pior**: usa o menor valor rolado
- Exibição da taxa de acerto
- Gráfico de distribuição dos resultados
- Gráfico comparando sucessos e falhas

## Possíveis usos

Este projeto pode ser usado para:

- analisar a chance de sucesso de testes em sistemas de RPG
- comparar diferentes mecânicas de rolagem
- balancear regras caseiras
- testar impacto de bônus e dificuldades
- apoiar criação de sistemas próprios
- ajudar mestres e jogadores a entender melhor probabilidades

## Tecnologias utilizadas

- Python
- Streamlit
- Matplotlib
